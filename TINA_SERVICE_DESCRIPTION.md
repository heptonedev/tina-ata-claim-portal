# TINA 서비스 소개 (에어드롭·랜딩용)

> **용도**: 에어드롭 웹페이지, 캠페인 랜딩, 공지 등에 그대로 복사·편집해 사용할 수 있는 설명 문구입니다.

---

## 한 줄 요약

**TINA**는 지도 위에서 **장소를 발견·등록**하고, **미션에 참여**해 **비스킷(리워드)**을 모은 뒤 **토큰 등으로 활용**할 수 있는 **위치 기반 리워드 플랫폼** 앱입니다. (Android / iOS)

---

## TINA란?

TINA는 일상 속 이동과 방문을 **데이터로 남기고 보상으로 연결**하는 서비스입니다. 사용자는 지도에서 주변 장소를 확인하고, 없는 장소는 직접 등록해 커뮤니티 지도를 함께 채울 수 있습니다. 주행·방문 인증·사진 촬영 등 **다양한 미션**에 참여하면 비스킷이 적립되며, 이를 **지갑·토큰 전환** 등과 연결해 활용할 수 있습니다.

단순한 포인트 적립을 넘어, **장소 정보의 정확성을 함께 다듬는 감별(퀴즈)** 같은 활동을 통해 **참여형 지도 경험**을 제공합니다.

---

## 주요 기능

| 영역 | 설명 |
|------|------|
| **지도** | MapLibre 기반 지도에서 장소를 탐색하고, 클러스터·상세 정보를 확인합니다. |
| **장소 등록** | 지도에 없는 장소를 제보·등록하고, 활동에 따라 비스킷 등 보상을 받을 수 있습니다. |
| **미션** | 주행 완료, 방문 체크인, 사진 미션 등 **위치 기반 미션**에 참여합니다. |
| **감별왕 (퀴즈)** | 등록된 장소 정보가 맞는지 투표·감별에 참여하고, 랭킹·게이미피케이션 요소를 즐깁니다. |
| **지갑·리워드** | 비스킷 적립, **TINA 토큰 지갑** 연동, 전환·이벤트(룰렛 등)와 연계될 수 있습니다. |
| **알림·딥링크** | 푸시·원링크 등을 통해 특정 화면(지도, 지갑, 미션, 공지 등)으로 안내할 수 있습니다. |

---

## 이용 흐름 (사용자 관점)

1. **앱 설치** 후 계정을 연결합니다.  
2. **지도**에서 관심 장소를 찾거나 **장소를 등록**합니다.  
3. **미션**을 수행하거나 **감별**에 참여해 **비스킷**을 쌓습니다.  
4. **지갑**에서 리워드·토큰 관련 기능을 확인하고, 캠페인·에어드롭 조건에 맞게 참여합니다.

## TINA 토큰·블록체인

**TINA 토큰**은 **Solana** 상에서 발행·전송되며, **SPL Token-2022**(Token Extensions) 프로그램을 사용합니다. 앱 내 지갑·외부 지갑 연동 시에도 **Solana 메인넷** 기준으로 동작합니다.

| 항목 | 내용 |
|------|------|
| **네트워크** | Solana (Mainnet-Beta 등, 서비스 정책에 따름) |
| **토큰 표준** | SPL **Token-2022** |
| **Token-2022 프로그램 ID** | `TokenzQdBNbLqP5VEhdkAS6EPFLC1PHnBqCXEpPxuEb` |

**토큰 Mint 주소**는 에어드롭·거래소·공식 공지에 게시된 값을 사용하세요. (본 문서에는 고정값을 넣지 않았습니다. 필요 시 공식 채널 값으로 채워 넣으면 됩니다.)
